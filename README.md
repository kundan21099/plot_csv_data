# plot_csv_data

***Description***

CSV of corresponding data need to be uploaded on UI to plot the repective data. Moreover, standard deviationa and mean values are also plotted alongwith statisctics at the end.

This project is still in developing stage. Further developments to generalize the plotting and converting log data to CSV files are not added yet.

**csv_to_plot_customized_for_phyphox** branch contains specific requirement of uplading time.csv (from Phyphox app) to plot the graph and consist of accelerometer and Gyroscope specific conditions for titles. .
